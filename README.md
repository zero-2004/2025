# Button-Driven FAQ Chatbot

## Overview
This project is a web-based FAQ chatbot designed to provide quick and structured responses using a button-driven interface. Instead of typing queries, users can select predefined options, making interaction simple and efficient.

## Features
- Interactive button-based user input (no typing required)
- Categorized FAQ sections (Services, Rates, Shipment, etc.)
- Dynamic responses using JavaScript
- Quick reply buttons for smooth navigation
- Typing indicator for better user experience
- Responsive chat UI with open/close toggle
- Scrollable chat interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. User clicks the chat button to open the chatbot
2. Main menu with categories is displayed
3. User selects a category
4. Chatbot shows related questions
5. User selects a question to get an answer
6. Additional options or navigation buttons are provided

## Project Structure
index.html # Structure of the chatbot UI
style.css # Styling and layout
script.js # Chatbot logic and interaction


## Key Components
- FAQ Data Structure: Stores categories, questions, and answers
- Dynamic UI Rendering: Messages and buttons generated using JavaScript
- Event Handling: Handles user interactions and navigation
- Chat Widget: Floating chatbot with toggle functionality

## Installation & Usage
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Click the chat button to start interacting
